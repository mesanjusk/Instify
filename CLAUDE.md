# Instify Project Instructions

## Git Workflow
- After every commit, always push to **both** the feature branch (`claude/wonderful-hamilton-wr39r8`) **and** `main`.
- Push to main immediately after committing — never wait for the user to ask.
- Command: `git push origin main`

## Stack
- Frontend: React + Vite + MUI (Material UI) — `SocialFront/`
- Backend: Express.js + MongoDB/Mongoose — `SocialBackend/`
- Live backend: `https://instify-kfpm.onrender.com`
- Live frontend: `https://app.instify.in`
